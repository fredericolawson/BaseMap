# BaseMap

A clean and simple Airtable schema visualization tool. BaseMap helps you understand and document your Airtable base structure by providing a clear visualization of tables, fields, and relationships.

## Features

- 🔑 Simple authentication with Airtable Personal Access Token
- 📊 Clear visualization of base schema
- 🔗 Relationship mapping between tables
- 📋 Copy schema as JSON
- 🎨 Clean, minimal interface
- 🌙 Dark mode support

## Getting Started

### Prerequisites

- Node.js 18.0 or later
- npm or yarn
- Airtable Personal Access Token
- Airtable Base ID

### Installation

1. Clone the repository:
```bash
git clone https://github.com/fredericolawson/BaseMap.git
cd BaseMap
```

2. Install dependencies:
```bash
npm install
# or
yarn install
```

3. Run the development server:
```bash
npm run dev
# or
yarn dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.

### Usage

1. Get your Airtable Personal Access Token from your account settings
2. Find your Base ID (it starts with 'app' and can be found in the API documentation)
3. Enter these credentials in BaseMap
4. View your base schema with tables, fields, and relationships
